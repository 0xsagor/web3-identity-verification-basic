# Web3 Identity Verification Basic

This project helps beginners learn how decentralized identity systems work using blockchain.

You will learn:
- What decentralized identity is
- How to connect a wallet
- How to store identity hash on-chain
- How to verify identity data
- How Web3 trust systems work

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-identity-verification-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- identity.js = Identity contract functions  
- web3.js = Blockchain connection  
- deploy.js = Contract deploy script  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- Save identity hash  
- Verify identity hash  
- Show verification result  
- Simple identity dashboard  

## Learning Goal
This project explains how blockchain can be used for decentralized identity systems.
